# Omni360 Product Brain

Версия с Supabase, настраиваемыми статусами и приоритетами.

## Установка

1. Выполнить `setup.sql` в Supabase SQL Editor.
2. Проверить URL и publishable key в `config.js`.
3. Загрузить все файлы в корень GitHub-репозитория.
4. Дождаться deploy Render и обновить страницу через Cmd+Shift+R.
5. Войти и нажать «Импортировать исходные реквесты».

## Статусы

Базовые: Новый, Обработать, Нужно уточнить, В YouTrack, Сделано, Отклонено.
Новые статусы можно добавлять в левой панели.

## Приоритеты

Не задан, Низкий, Средний, Высокий, Критический.

Старое поле качества сохраняется в `legacy_quality` и отображается в карточке только для справки.

## Теги

В карточке реквеста можно выбрать несколько существующих тегов или создать новый тег на ходу.
Верхний фильтр показывает реквесты с выбранным тегом, а общий поиск также ищет по названиям тегов.

## Ручное добавление

Кнопка «Новый реквест» создаёт запись со статусом «Новый» и сразу открывает её для заполнения.
Исходное название и формулировку можно редактировать вместе с продуктовыми полями.
